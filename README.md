## Açma / Çalıştırma

1. Unity Hub → **Add** (ya da **Open**) → proje klasörünü seç.
2. Unity Editor açılınca **Play** ile çalıştır.

## Repo Notları

- Git’te tutulan klasörler: `Assets/`, `Packages/`, `ProjectSettings/`
- Git’te tutulmayanlar (otomatik üretilir): `Library/`, `Temp/`, `Logs/`, `obj/`, `.vs/`
- Gizli dosyalar (keystore vb.) asla GitHub’a gönderilmez; `.gitignore` bunu engeller.
